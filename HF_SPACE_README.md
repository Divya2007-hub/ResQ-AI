# ResQ AI — Multi-Agent Disaster Response & Emergency Assistant

**Track:** Agents for Good — Kaggle AI Agents Competition

When disaster strikes, ResQ AI's team of specialist AI agents triages, plans, and coordinates a personalised emergency response in seconds — and shows you exactly how they reasoned to get there.

## How to Use

1. Click the app link above
2. Fill out the emergency form
3. Watch the multi-agent reasoning unfold in real time
4. Get your prioritised action plan with PDF download

## Environment Variables

Set these in your Space's **Settings → Repository Secrets**:

- `GEMINI_API_KEY` — Your Google Gemini API key
